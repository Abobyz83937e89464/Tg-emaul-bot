# Tg-emaul-bot

Telegram Mini App для автоматической регистрации почтовых аккаунтов.

## Возможности

- Регистрация почт на популярных сервисах
- Защита от обнаружения ботами
- CD 2 часа между регистрациями  
- Web App интерфейс в Telegram
- База данных Supabase

## Поддерживаемые сервисы

- Outlook.com (без номера)
- Yahoo Mail (без номера)
- Mail.com (без номера)
- ProtonMail (без номера)

## Технологии

- Python + FastAPI
- Telegram Bot API
- Supabase (PostgreSQL)
- Playwright для автоматизации

## Установка

1. Клонируй репозиторий
2. Установи зависимости: `pip install -r requirements.txt`
3. Настрой переменные окружения в `config.py`
4. Запусти: `python main.py`
